# Syntecxhub_Notes_App_Backend

📌 Project Overview
This project is a Notes App Backend API developed using Node.js, Express.js, and MongoDB. It allows users to register, login, and manage their personal notes securely. Each note is associated with a specific user, ensuring that only the owner can access and modify their notes.

The project demonstrates backend development concepts such as RESTful APIs, authentication, database relationships, and CRUD operations.

🚀 Features
User Registration
User Login with JWT Authentication
Create Notes
Get All Notes for Logged-in User
Get Single Note
Update Note
Delete Note
Archive (Soft Delete) Notes
Notes linked to Users
Mongoose populate() to fetch user details with notes
Protected Routes using Middleware
